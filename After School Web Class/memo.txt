HTML 안에서 스타일을 적용하는 방식 -> 인라인 방식

태그 선택 - 태그명{}
color - 폰트색
아이디 선택 - 앞에 #을 쓰고 아이디를 입력
background-color - 배경색
클래스 선택자 - 앞에 .을 쓰고 클래스를 입력
자손 선택자 - 띄어쓰기를 한 경우에는 모든 자손이 선택됨
자식 선택자 - > 기호를 사용한 경우 바로 아래 자식이 선택됨
특정 태그의 클래스의 바로 밑에 태그 선택 - 태그명.클래스명 > 태그명
전체 선택자 - *
font-family - 글꼴
인접 형제 선택자(결합자) - +,  첫번째 요소의 바로 뒤에 위치하면서 같은 부모를 공유하는 두 번째 요소를 선택
p태그 중에서 클래스를 갖고 있는 애들 전부 선택 - 태그명[클래스명]
글씨 크기 - font-size
다중 선택 - ,로 구분
선과 관련된 속성 - border 굵기 실선/점선 색상
ul>(li>a[href="#"])*3 => 3개 생성
text-decoration 텍스 밑에 밑줄 등
마우스를 올렸을 때 글씨에 특정 스타일을 적용하는 방법 - 태그:hover
